请注意：
1. 如果正常安装了JAVA运行时环境(JRE)，可以直接双击运行Clock_1.0.jar 文件， 即可运行游戏程序；或者在Clock_1.0.jar 文件上单击鼠标右键，并选择打开方式为 "Java(TM) 2 Platform Standard Edition binary"，也可以将游戏程序运行起来。

2. 如果JAVA运行时环境(JRE)不是正常安装的，但可以正常使用java.exe命令，那么可以双击Clock_1.0.bat 文件，也可以立即将游戏程序运行起来。

3. 如果没有JAVA运行时环境(JRE),可以到Oracle网站下载并安装。

4. 本游戏运行在JVM中，一般不会对您的系统造成严重影响（如系统崩溃 ^_^ 等），请放心使用。
